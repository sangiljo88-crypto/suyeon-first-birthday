### 💡 사진 준비 팁

**사진이 10장이 안 될 때:**
- 같은 사진을 여러 번 복사해서 넣으세요
- 예: `photo-1.jpg`를 복사해서 `photo-2.jpg`, `photo-3.jpg`로 이름 변경

**사진 이름 바꾸는 법:**
1. 사진 선택
2. 마우스 우클릭 → "이름 바꾸기" (또는 F2 키)
3. `photo-1.jpg`로 변경

**사진 크기 줄이는 법 (선택사항, 로딩 속도 향상):**
- 온라인 도구: [iloveimg.com](https://www.iloveimg.com/ko/resize-image)
- 권장 크기: 가로 1200px 이하
- 포맷: JPG 권장 (PNG보다 용량 작음)

## 🎵 4단계: 음악 넣기

```
수연이돌잔치/
├── baby_milestone.html
├── cover-photo.jpg
├── birth-photo.jpg
├── background-music.mp3  ⬅️ 음악 파일 (이름 정확히!)
└── month-0/ ~ month-12/
```

### 음악 파일 준비

1. **음악 다운로드**
   - YouTube Audio Library: [링크](https://www.youtube.com/audiolibrary)
   - Free Music Archive: [링크](https://freemusicarchive.org/)
   - 또는 직접 녹음한 파일

2. **파일명 변경**
   - 반드시 `background-music.mp3`로 이름 변경
   - MP3 형식이 가장 안정적

3. **같은 폴더에 배치**
   - HTML 파일과 같은 위치에 넣으세요

### 💡 음악 추천
- 클래식 동요 (피아노 버전)
- 오르골 사운드
- 잔잔한 인스트루멘탈
- 길이: 3~5분 적당

## 🎬 5단계: 유튜브 영상 넣기

영상은 파일이 아니라 **유튜브 링크**를 사용합니다!

### 유튜브에 영상 업로드

1. [YouTube Studio](https://studio.youtube.com/) 접속
2. "만들기" → "동영상 업로드"
3. 영상 업로드
4. **공개 설정**: "일부 공개" 또는 "공개" 선택 (비공개는 안됨!)
5. 업로드 완료

### 영상 ID 확인

업로드 후 영상 URL:
```
https://www.youtube.com/watch?v=dQw4w9WgXcQ
                                 ^^^^^^^^^^^^
                                 이 부분이 VIDEO_ID
```

### HTML에 영상 ID 입력

1. `baby_milestone.html`을 **메모장** 또는 **텍스트 에디터**로 엽니다
   - 마우스 우클릭 → "연결 프로그램" → "메모장"

2. `Ctrl + F` 눌러서 `VIDEO_ID` 검색

3. 찾은 부분:
```html
<iframe 
    src="https://www.youtube.com/embed/VIDEO_ID"
```

4. `VIDEO_ID`를 실제 영상 ID로 교체:
```html
<iframe 
    src="https://www.youtube.com/embed/dQw4w9WgXcQ"
```

5. 저장 (`Ctrl + S`)

## 🧪 7단계: 로컬에서 테스트

1. `baby_milestone.html` 파일을 **더블클릭**
2. 브라우저에서 열림
3. 확인 사항:
   - ✅ 커버 사진이 보이나요?
   - ✅ 월별 슬라이더가 작동하나요?
   - ✅ 음악이 재생되나요?
   - ✅ 유튜브 영상이 보이나요?
   - ✅ 방명록이 작동하나요?

## 🎨 추가 팁

### 사진 일괄 이름 바꾸기 (Windows)

1. 폴더에 사진 10장 넣기
2. 첫 번째 사진 선택 → `F2` → `photo-1` 입력 후 엔터
3. 나머지 사진도 순서대로 선택해서 이름 변경

### 사진 일괄 이름 바꾸기 (Mac)

1. 사진 10장 모두 선택
2. 우클릭 → "10개 항목의 이름 변경"
3. "텍스트 바꾸기" 선택
4. 형식: `photo-` + 숫자

### 빠른 폴더 생성 (Windows)

배치 파일 만들기:
1. 메모장 열기
2. 아래 내용 복사:
```batch
mkdir month-0
mkdir month-1
mkdir month-2
mkdir month-3
mkdir month-4
mkdir month-5
mkdir month-6
mkdir month-7
mkdir month-8
mkdir month-9
mkdir month-10
mkdir month-11
mkdir month-12
```
3. `폴더생성.bat`로 저장
4. 더블클릭 실행 → 13개 폴더 자동 생성!
