## 📊 파일 개수 정리

- HTML 파일: 1개
- 기본 사진: 2개
- 음악 파일: 1개
- 월별 폴더: 13개
- 월별 사진: 130개 (13개월 × 10장)
- **총 파일**: 134개
- **총 폴더**: 14개 (프로젝트 폴더 + 13개 월별 폴더)

- 폴더 구조 만들기: 10분

- 음악 준비: 30분
- 유튜브 영상 업로드: 30분

- 테스트: 20분
- 배포: 20분

**총 예상 시간: 3~4시간**

## 💡 시간 단축 팁

1. **사진이 많지 않다면**: 같은 사진을 여러 번 복사해서 사용
2. **일괄 이름 변경**: 파일 탐색기에서 여러 파일 선택 후 F2
3. **배치 파일 사용**: 폴더 자동 생성 스크립트 활용
4. **가족에게 도움 요청**: 사진 정리 작업은 나눠서 진행
