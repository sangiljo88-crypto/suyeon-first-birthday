# ✅ GitHub + Netlify 배포 체크리스트

## 준비 단계
- [ ] 프로젝트 폴더에 모든 파일 준비 완료
- [ ] 로컬에서 테스트 완료 (HTML 더블클릭)

---

## GitHub 설정 (5단계)

### 1. GitHub 계정
- [ ] GitHub.com 가입 완료
- [ ] 이메일 인증 완료

### 2. GitHub Desktop 설치
- [ ] desktop.github.com에서 다운로드
- [ ] 설치 완료
- [ ] GitHub 계정으로 로그인

### 3. Repository 생성
- [ ] File → New repository
- [ ] Name: `suyeon-first-birthday` (예시)
- [ ] Initialize with README 체크
- [ ] Create repository 클릭

### 4. 파일 복사
- [ ] 생성된 GitHub 폴더 열기
- [ ] 준비한 모든 파일 복사 (Ctrl+A, Ctrl+C, Ctrl+V)
- [ ] HTML, 사진, 음악 파일 모두 확인

### 5. 업로드 (Push)
- [ ] GitHub Desktop에서 변경사항 확인
- [ ] Summary 입력: "첫 번째 업로드"
- [ ] "Commit to main" 클릭
- [ ] "Push origin" 클릭
- [ ] GitHub.com에서 파일 확인

---

## Netlify 배포 (4단계)

### 1. Netlify 계정
- [ ] Netlify.com 접속
- [ ] Sign up with GitHub
- [ ] Authorize Netlify
- [ ] 이메일 인증

### 2. 새 사이트 만들기
- [ ] "Add new site" 클릭
- [ ] "Import an existing project" 선택
- [ ] "Deploy with GitHub" 클릭

### 3. Repository 연결
- [ ] Repository 목록에서 선택
  - 안 보이면: Configure → Only select repositories → 선택 → Save
- [ ] "suyeon-first-birthday" 선택

### 4. 배포 설정
- [ ] Branch: main 확인
- [ ] Build settings 비워두기 (HTML은 빌드 불필요)
- [ ] "Deploy site" 클릭
- [ ] 배포 완료 대기 (1-3분)

---

## 확인 및 공유 (3단계)

### 1. 사이트 테스트
- [ ] Netlify URL 클릭
- [ ] 커버 사진 확인
- [ ] 월별 슬라이더 작동 확인
- [ ] 음악 재생 확인
- [ ] 유튜브 영상 확인
- [ ] 방명록 테스트

### 2. 도메인 변경 (선택)
- [ ] Site settings → Change site name
- [ ] 원하는 이름 입력
- [ ] Save

### 3. QR 코드 생성
- [ ] qr-code-generator.com 접속
- [ ] Netlify URL 입력
- [ ] QR 코드 다운로드
- [ ] 답례품 카드에 인쇄
